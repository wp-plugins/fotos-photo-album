Spanish documentation and last version:

You can share your Fotos.es albums in your blog.
Include and show your favourite Fotos.es photos and albums inside your blog with this easy-to-install plugin

== Installation ==

Spanish documentation and last version:
http://download.fotos.es/

Installation process is very easy:

1. Upload `fotos.es` to '/wp-content/plugins/' directory
2. Activate the plugin under 'Plugins' menu in WordPress
3. Create or edit a post, click on blue camera icon and search albums.
